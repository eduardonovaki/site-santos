# Site do Santos FC

Projeto de site inspirado no Santos Futebol Clube, desenvolvido para praticar HTML, CSS e JavaScript, com foco em layout responsivo e interatividade.
- Layout inspirado em site esportivo
- Página responsiva (mobile e desktop)
- Interações com JavaScript
- Organização de conteúdo (jogadores, história, etc.)

## Tecnologias:
- HTML5
- CSS3
- JavaScript

## Objetivo
Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento web front-end, aplicando conceitos desenvolvidos no Nano-Course de Front-end da FIAP.
